# dbmini – Agriculture & Resource Management Portal

A web-based system to empower farmers and workers with access to government schemes, crop tracking, and resource management tools.

## Features
- Farmer & Worker registration and login
- Role-based dashboards
- Government scheme listings
- Crop and equipment tracking
- Secure authentication

## Tech Stack
- PHP, MySQL, HTML/CSS
- XAMPP (Apache + MySQL)
